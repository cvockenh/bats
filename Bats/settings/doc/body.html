Introduction to BATS

BATS is designed to automatically calculate standard and blank corrected results for measured samples. It is an easy to use data reduction tool that requires minimal user input.

Because the program is designed to automatically calculate and update the results the quality and status of the acquired data can be monitored quasi-online. This allows us to see, if the desired precision including the complete error propagation has been reached.
