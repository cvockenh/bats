Procedure of data calculations

The data is stored in an Array of runs. A run is a single measurement. Typically 3-10 runs are measured of one sample.

The procedure is as follows:
Initialisation of data (initData):

This is called only once, after runs are imported.

  • Runs are sorted (-> label and run sorted)
  • List of corrections is initialised
  • Runs are initialised (initRunCalc), no real calculations jet.
  • Sample list is created (from label sorted run list (initSampleCalc)
Calculate data (calcAll)

This is called always, when data is (re-)calculated.

  • Preferences are loaded (current correction, isobar correction etc)
  • Runs are pre-calculated (with current correction and isobar correction, but without standard and blank correction, runPreCalc)
  • Samples are pre-calculated (samplePreCalc)
  • Corrections are calculated (setCorrection):
    - Isobar and time correction of radioisotope
    - Blank correction (on ratio)
    - ∆13C standard correction (for radiocarbon only)
    - Standard correction
  • Corrected ratios of runs and samples are calculated


A more detail description what BATS does can be found in Wacker et al. 2010.
