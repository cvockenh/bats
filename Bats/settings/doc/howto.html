How-to

  • Import the data either directly from the database (-> Import from database), from a dat-file (format for dbxl-import) or from a bats-file previously created.
  • Check if you need to split your dataset for the corrections.
  • Check that the errors and offsets of the current measurement are set right (-> Preferences).
  • (Check if you need a time correction on radioisotope.)
  • Set appropriate standard, blank (and isobar) labels (if not jet done).
  • Make current correction, if needed (-> current correction panel).
  • Make background or isobar correction (open isobar correction panel).
  • Check blank calcuation; the mean blank with the uncertainty of the blank scatter should should be taken. Alternatively, the mean with only the statistical error can be taken, when in preferences the appropriate blank scatter is set.
  • For radiocarbon the 13/12 correction has to be calculated. Check for outlayers and check if errors are right (often the current measurement errors are underestimated and need to be adjusted now!)
  • Make radioisotope ratio correction. Check again for outlayers. Internal and external errors should be about the same!
    In case the sigma is significantly larger, there is a dominant non-statistic uncertainty on the standard data. You might choose then the scatter of the runs (error column) for the error of the standard...
  • Now check the final data. I might be you are finished!
