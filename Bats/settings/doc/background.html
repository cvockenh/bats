Background and isobar correction

In Preferences a background isotope can be selected. The isotope is multipied with the background correction factor (can be set either in Preferences or with the Isobar correction panel) and subtracted from the radioisotope. The correction factor error is relative.

For 14C the blanks are automatically used to plot the isobar correction.

For 10Be and other isotopes the samples used for the isobar correction must have the sampletype iso! The obtained correlation is automatically taken, once the plot is done (needs to be improved).
