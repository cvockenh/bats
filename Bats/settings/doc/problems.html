Known Problems

  • BATS needs a lot of memory -> the Java virtual machine has to be started with an increased maximum heap memory size. In the console the Program can be started with:
    java -jar bats.jar -Xmx400M
    There is now as well an executable .bat file to start Bats with more memory on Windows!
