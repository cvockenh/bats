NEC files

For the import the directory with

 

The blank is subtracted from standards and samples. This is normally not a processing blank (except the standard is porcessed in the same way)!

The blank subtraction is done after the isobar correction!

Samples with sampletype b, bl and blank are taken as blank (not case sensitive).

 

For Radiocarbon a first simple version of a blank correction for a constant contamination (and for constant background) is implemented as well (Corrections > Constant contamination). However, the amount of contamination and its measured ratio has to be known and the sample weight must be given!

 

Attention: If there is NO blank, the blank is set 0!

 

The blank is NOT subtracted from the blank itself and from samples with sampletype nb.
